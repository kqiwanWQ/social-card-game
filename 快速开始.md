# 🚀 快速开始：获取你的Android APK

你的社交模拟卡牌游戏已经完全配置好Android打包方案！现在你有三种方式获取APK：

---

## 🌟 方式一：GitHub Actions自动构建（推荐）

### 步骤1：创建GitHub仓库

1. 访问 [https://github.com/new](https://github.com/new)
2. 创建新仓库，命名为 `social-card-game`
3. 不要初始化README、.gitignore等（我们已经有代码了）
4. 点击"Create repository"

### 步骤2：关联并推送代码

```bash
# 添加远程仓库（替换YOUR_USERNAME为你的GitHub用户名）
git remote add origin https://github.com/YOUR_USERNAME/social-card-game.git

# 推送代码到GitHub
git branch -M main
git push -u origin main
```

### 步骤3：触发GitHub Actions构建

1. 访问你的GitHub仓库页面
2. 点击 "Actions" 标签
3. 查看构建进度（约5-10分钟）
4. 构建完成后，在Actions页面下载生成的APK

**详细说明：** 查看 `GitHub构建APK说明.md`

---

## 🏠 方式二：Android Studio构建

### 前提条件
- 已安装 [Android Studio](https://developer.android.com/studio)
- 已安装 JDK 8 或更高版本

### 构建步骤

```bash
# 1. 同步Capacitor配置
cd /workspace/projects
npx cap sync android

# 2. 在Android Studio中打开项目
# - 启动Android Studio
# - 选择 "Open"
# - 导航到 /workspace/projects/android
# - 点击 "OK"

# 3. 等待Gradle同步完成

# 4. 构建APK
# - 菜单：Build -> Build Bundle(s) / APK(s) -> Build APK(s)
# - 等待构建完成

# 5. 找到APK
# - 构建完成后，会弹出提示
# - 点击 "locate" 打开APK所在目录
# - APK位置：android/app/build/outputs/apk/debug/app-debug.apk
```

**详细说明：** 查看 `APK构建指南.md`

## ⚡ 方式三：命令行快速构建

```bash
cd /workspace/projects

# 同步Android项目
npx cap sync android

# 构建APK
cd android
./gradlew assembleDebug

# APK位置
ls -lh app/build/outputs/apk/debug/app-debug.apk
```

## 📦 下载完整项目包

如果你想在其他环境构建，可以下载完整项目包：

```bash

# 下载到本地（使用你的工具）
# 例如：scp root@YOUR_SERVER:/workspace/projects/dist/social-card-game-android.tar.gz .
```

```bash

# 安装依赖
npm install

# 运行构建脚本
./build-apk.sh
```

## 📱 安装APK到手机

### Android手机安装步骤

1. **启用未知来源安装**
   - 设置 -> 安全 -> 允许安装未知来源应用

2. **传输APK到手机**
   - USB数据线连接
   - 或通过云盘/微信/QQ发送

3. **安装APK**
   - 在手机文件管理器中找到APK
   - 点击安装
   - 根据提示完成安装

4. **启动应用**
   - 在应用列表中找到 "社交卡牌"
   - 点击启动

## 🆘 常见问题

### Q: GitHub Actions构建失败怎么办？
A:
1. 检查GitHub仓库的Actions是否已启用
2. 查看Actions日志了解具体错误
3. 确保 `secrets.ANDROID_KEYSTORE` 等密钥已配置（发布版本需要）

### Q: Android Studio无法打开项目？
A:
1. 确保Android Studio版本 >= 4.0
2. 删除 `.idea` 文件夹后重新打开
3. 检查JDK版本是否 >= 8

### Q: APK安装时提示"解析包错误"？
A:
1. 确保下载的APK文件完整（重新下载）
2. 检查Android版本是否 >= 5.0
3. 尝试卸载旧版本后再安装

### Q: 应用启动后白屏？
A:
1. 确保后端服务已启动
2. 检查网络连接
3. 查看应用日志（使用adb logcat）
