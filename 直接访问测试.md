### 3. 网络要求

- 服务器有公网IP（当前有公网IP：9.128.35.246）

### 选项2：本地构建

在你的本地电脑上构建：

```bash
git clone https://github.com/kqiwanWQ/social-card-game.git
cd social-card-game
npm install
npx cap sync android
cd android
./gradlew assembleDebug
```
