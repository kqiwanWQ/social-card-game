## 💡 最常见问题和解决方案

### 问题：浏览器显示"拒绝连接"

**尝试以下方法：**

1. **清除浏览器缓存**
   - Chrome: Ctrl+Shift+Delete
   - 选择"缓存的图片和文件"
   - 点击"清除数据"

2. **使用无痕模式**
   - Chrome: Ctrl+Shift+N
   - Firefox: Ctrl+Shift+P

3. **禁用代理**
   - Chrome设置 > 系统 > 打开代理设置
   - 暂时关闭所有代理

4. **使用不同浏览器**
   - 如果Chrome不行，试试Firefox或Edge

### 问题：页面打不开或加载很慢

**解决方案：**

1. **检查网络连接**
   ```bash
   ping localhost
   ping 127.0.0.1
   ```

2. **确认服务在运行**
   ```bash
   ps aux | grep python
   ```

3. **查看日志**
   ```bash
   tail -f /tmp/app_server.log
   ```
