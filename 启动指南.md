## 🚀 立即开始体验

### 方式1：使用启动脚本（最简单）

**Linux/Mac:**
```bash
cd /workspace/projects
./scripts/start.sh
```

**Windows:**
```bash
cd /workspace/projects
scripts\start.bat
```

脚本会自动启动服务，并提供访问链接。

### 方式2：手动启动

```bash
cd /workspace/projects
python src/server.py
```

等待看到以下提示表示启动成功：
```
INFO:     Started server process [xxx]
INFO:     Waiting for application startup.
INFO:     Application startup complete.
INFO:     Uvicorn running on http://0.0.0.0:8000 (Press CTRL+C to quit)
```

## 📋 验证服务状态

### 检查服务是否运行
```bash
# 检查端口8000是否被占用
lsof -i :8000
```

如果看到类似输出，说明服务正在运行：
```
COMMAND   PID USER   FD   TYPE DEVICE SIZE/OFF NODE NAME
python3  439 root    8u  IPv4  xxxxx      0t0  TCP *:8000 (LISTEN)
```

### 测试API接口
```bash
curl http://localhost:8000/api/cards
```

应该返回JSON格式的卡牌列表。

## 🐛 如果还是打不开

### 1. 检查服务是否启动
```bash
ps aux | grep "python src/server.py"
```

如果没有输出，说明服务未启动，请重新启动。

### 2. 检查端口是否被占用
```bash
lsof -i :8000
```

如果端口被其他程序占用，可以：
- 停止占用端口的程序
- 或者修改server.py中的端口号

### 3. 检查防火墙设置
确保防火墙允许访问8000端口。

### 4. 使用正确的地址
- ✅ 正确：http://localhost:8000/
- ✅ 正确：http://localhost:8000/app.html
- ❌ 错误：http://localhost:8000
- ❌ 错误：file:///workspace/projects/assets/app.html

## 📱 在手机上访问

### 方法1：在同一WiFi网络下

1. **查看电脑IP地址**
   ```bash
   # Linux/Mac
   ifconfig | grep "inet "

   # Windows
   ipconfig
   ```

2. **在手机浏览器中访问**
   ```
   http://YOUR_PC_IP:8000/app.html
   ```

   例如：如果电脑IP是192.168.1.100
   ```
   http://192.168.1.100:8000/app.html
   ```

### 方法2：使用Chrome移动端模拟器

1. 在电脑浏览器中打开 http://localhost:8000/app.html
2. 按F12打开开发者工具
3. 点击设备工具栏图标（Ctrl+Shift+M）
4. 选择移动设备（如iPhone 12 Pro）
