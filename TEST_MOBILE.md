# 移动端体验测试指南

## 📱 测试方式

### 方式1：浏览器移动端模拟器

1. **打开Chrome开发者工具**
   - 按F12或右键选择"检查"
   - 点击设备工具栏图标（Ctrl+Shift+M）

2. **选择移动设备**
   - 选择"iPhone 12 Pro"或"Samsung Galaxy S20"
   - 或者输入自定义尺寸（375x812）

3. **访问应用**
   ```
   http://localhost:8000/app.html
   ```

4. **测试功能**
   - ✨ 点击"新增卡牌人物"
   - 📝 填写表单并保存
   - 🎯 点击卡牌查看详情
   - 💬 添加互动记录
   - 🎮 生成策略报告

### 方式2：真机测试

#### Android手机

1. **确保手机和电脑在同一网络**
   - 查看电脑IP地址：
     ```bash
     # Linux/Mac
     ifconfig | grep "inet "

     # Windows
     ipconfig
     ```

2. **修改API地址**
   - 编辑 `assets/app.html`
   - 将 `http://localhost:8000/api` 改为 `http://YOUR_PC_IP:8000/api`

3. **在手机浏览器中打开**
   ```
   http://YOUR_PC_IP:8000/app.html
   ```

#### iOS手机（iPhone）

1. **确保手机和电脑在同一网络**
2. **使用电脑IP地址访问**
3. **建议使用Safari浏览器**

### 方式3：打包成APK测试

1. **安装Android Studio**
2. **同步项目**
   ```bash
   npm run sync
   npm run android
   ```

3. **在Android Studio中**
   - 选择设备或模拟器
   - 点击运行按钮
   - 或生成APK文件安装到手机
