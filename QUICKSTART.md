# 🎮 社交模拟卡牌 - 快速入门

## 5分钟快速体验

### 方式1：直接使用（最简单）

1. **启动服务**
   ```bash
   # Linux/Mac
   ./scripts/start.sh
   
   # Windows
   scripts\start.bat
   ```

2. **打开应用**
   - 在浏览器中打开: http://localhost:8000/app.html
   - 建议使用手机浏览器或Chrome开发者工具的移动端模式

3. **开始使用**
   - 点击「✨ 新增卡牌人物」创建第一个卡牌
   - 记录与卡牌的互动
   - 点击「🎯 策略报告」获取AI分析

### 方式2：开发模式

1. **启动后端**
   ```bash
   python src/server.py
   ```

2. **访问应用**
   - Web版: http://localhost:8000/
   - 移动端版: http://localhost:8000/app.html

### 方式3：打包成APP（推荐）

1. **安装依赖**
   ```bash
   npm install
   ```

2. **同步Capacitor**
   ```bash
   npm run sync
   ```

3. **在手机上运行**
   ```bash
   # Android
   npm run android
   
   # iOS (需要Mac)
   npm run ios
   ```

详细打包教程请参考 [README_MOBILE_APP.md](README_MOBILE_APP.md)

## 💡 使用技巧

### 1. 如何获取更好的策略报告？
- **丰富人物信息**：尽可能详细地填写人物的性格、能力、简介等信息
- **记录多条互动**：添加多条互动记录，让AI更了解你们的关系
- **定期更新**：随着关系的发展，定期更新互动记录和生成新报告

### 2. 如何高效管理卡牌？
- **使用搜索功能**：快速找到特定的卡牌
- **合理分类**：可以通过职业、关系类型等维度组织卡牌
- **定期维护**：定期查看和更新卡牌信息

### 3. 何时生成策略报告？
- **关系初期**：了解如何建立良好的关系
- **关系发展期**：获得深化关系的建议
- **遇到困难时**：寻找解决方案
- **定期回顾**：定期生成报告追踪关系发展

## ⚙️ 配置说明

### 修改API地址

编辑 `assets/app.html` 文件，找到以下代码：

```javascript
const API_BASE = 'http://localhost:8000/api';
```

根据你的部署环境修改：
- **本地开发**：`http://localhost:8000/api`
- **局域网访问**：`http://YOUR_PC_IP:8000/api`
- **线上部署**：`https://your-domain.com/api`

### 自定义主题颜色

## 🐛 常见问题

### Q: 为什么打不开应用？
A: 请确保后端服务已启动，检查端口8000是否被占用。

### Q: 策略报告生成很慢？
A: 首次生成需要调用AI模型，可能需要10-15秒，请耐心等待。

### Q: 如何在手机上使用？
A: 
1. 确保手机和电脑在同一网络
2. 修改API地址为电脑的IP地址
3. 在手机浏览器中打开链接

### Q: 数据会丢失吗？
A: 数据存储在Supabase数据库中，不会因为关闭应用而丢失。

### Q: 可以导出数据吗？
A: 目前暂不支持导出功能，后续版本会添加。
